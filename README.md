# 🚀 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Maithali/Modern_React_UI.git
```

Move into project folder:

```bash
cd Modern_React_UI
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
